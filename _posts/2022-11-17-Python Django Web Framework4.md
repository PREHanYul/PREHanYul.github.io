# Python Django Web Framework4

### Routing URLConf

###### (1) Routing이란?

어떤 웹 프레임워크를 사용하던 간에

가장 먼저 하는 작업이 라우팅.

그래서 라우팅은 뭔데.

"네트워크에서 경로를 선택하는 프로세스"이다.

경로는 사용자가 접속한 경로이고 그 경로를 

누가 처리할 것인지 하는 것인 라우팅.



###### (2)라우팅이 중요한 이유는?

네트워크 통신의 효율성을 높인다.

라우팅은 네트워크가 정체 없이 최대한 많은 용량을

사용할 수 있도록 데이터 트래픽을 관리함으로써, 네트워크 장애를 최소화시켜준다.



###### (3)라우팅 유형

-정적 라우팅

-동적 라우팅
